# ROS 2 Publisher Practice

## What I studied today

Today, I practiced a basic ROS 2 Python publisher.

I learned how to:
- create and run a simple publisher node
- publish string messages to a topic
- check topics with `ros2 topic list`
- inspect messages with `ros2 topic echo /topic`

## Result

I successfully ran a minimal ROS 2 publisher that sent `Hello World` messages to `/topic`.
